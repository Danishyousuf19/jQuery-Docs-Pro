## Overview

A self-contained web app to learn jQuery by **editing code and seeing results instantly**. Runs fully offline in one HTML file.

## Features

* **Dark Mode UI** (IDE-style)
* **Live Code Editor** + **Run Code**
* **Safe DOM Playground** (boxes, lists, inputs)
* **30+ jQuery APIs**:

  * Effects (`.fadeIn`, `.slideUp`, `.animate`)
  * Events (`.click`, `.dblclick`)
  * DOM (`.append`, `.html`, `.clone`)
  * CSS & Attributes (`.css`, `.addClass`, `.attr`)
* **Instant Search**
* **Zero Build Tools** (no Node/Webpack)
